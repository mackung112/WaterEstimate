# Thai Language Only Directive

## Core Instruction
คุณ **ต้อง** สื่อสารด้วยภาษาไทย (Thai language) เท่านั้นในทุกๆ การโต้ตอบกับผู้ใช้ เนื่องจากผู้ใช้ได้ระบุอย่างชัดเจนว่าไม่สันทัดภาษาอังกฤษและไม่สามารถอ่านหรือแปลได้

## Rules (กฎเกณฑ์)
1. **การพูดคุยและคำอธิบาย (Chat and Explanations):** การตอบสนองด้วยภาษาธรรมชาติ คำอธิบาย และข้อความสนทนาทั้งหมดต้องเป็นภาษาไทย
2. **คอมเมนต์ในโค้ด (Code Comments):** คอมเมนต์ใดๆ ที่เขียนลงในโค้ด (ไม่ว่าจะเป็นโค้ดใหม่หรือการแก้ไขโค้ดเดิม) ควรเป็นภาษาไทย เพื่อให้ผู้ใช้อ่านเข้าใจได้ง่าย
3. **เอกสารและแผนงาน (Artifacts and Documentation):** ไฟล์ markdown, รายการงาน (task lists), แผนการทำงาน (implementation plans) หรือเอกสารใดๆ ที่สร้างเพื่อให้ผู้ใช้อ่าน ต้องเขียนเป็นภาษาไทย
4. **ข้อควรระวัง:** คุณสามารถใช้ภาษาอังกฤษได้เฉพาะในส่วนที่เป็นข้อจำกัดทางเทคนิคเท่านั้น เช่น ไวยากรณ์ของโค้ด (syntax), ชื่อตัวแปร (variable names), คำสั่งระบบ (bash commands) หรือพาธของไฟล์ (file paths) นอกเหนือจากนั้น คำอธิบายรอบข้างต้องเป็นภาษาไทยทั้งหมด

## Examples (ตัวอย่าง)
- ❌ **ไม่ถูกต้อง (Incorrect):** "I have updated the `index.html` file. The new feature is ready."
- ✅ **ถูกต้อง (Correct):** "ฉันได้อัปเดตไฟล์ `index.html` เรียบร้อยแล้วครับ ฟีเจอร์ใหม่พร้อมใช้งานแล้ว"
