## 1. ภาพรวมของระบบ

**WaterEstimate** (ระบบประมาณการออนไลน์) คือแอปพลิเคชันเว็บสำหรับการบริหารจัดการงานประมาณการติดตั้งประปา ครอบคลุมตั้งแต่การสร้างรายการงาน การบันทึกข้อมูลลูกค้า การคำนวณค่าใช้จ่ายอัตโนมัติ ไปจนถึงการออกรายงานสำหรับสั่งพิมพ์

### ความสามารถหลักของระบบ

| ความสามารถ | รายละเอียด |
|---|---|
| 📊 **ภาพรวมงาน (Dashboard)** | แสดงสถิติ, ค้นหา, กรอง และจัดเรียงรายการงานทั้งหมด |
| 📝 **บันทึกงานติดตั้ง (Form)** | กรอกข้อมูลเอกสาร, ลูกค้า, แผนที่, รูปแบบติดตั้ง, รายการวัสดุ และบุคลากร |
| 📦 **คลังวัสดุ (Materials)** | จัดการรายการวัสดุ ราคา และหมวดหมู่ |
| 👥 **จัดการบุคลากร (Personnel)** | จัดการข้อมูลผู้สำรวจ ผู้ตรวจสอบ และผู้อนุมัติ |
| 🖨️ **ออกรายงาน (Reports)** | พิมพ์รายงาน 3 ฉบับ (ใบประมาณการ, ใบแจ้งค่าใช้จ่าย, แบบแปลน) |
| 🗺️ **แผนที่ GPS (Leaflet)** | ปักหมุดพิกัดสถานที่ติดตั้งบนแผนที่ |
| 🖼️ **อัปโหลดรูปภาพ GIS** | อัปโหลดภาพจากระบบ GIS เก็บไว้บน Supabase Storage |

### เทคโนโลยีที่ใช้

- **Frontend:** HTML, JavaScript, CSS (Bootstrap 5, Font Awesome)
- **Backend/Database:** Supabase (PostgreSQL)
- **แผนที่:** Leaflet.js + OpenStreetMap
- **Font:** Sarabun (Google Fonts)

## 2. ข้อกำหนดเบื้องต้น

- **เว็บเบราว์เซอร์:** Google Chrome, Microsoft Edge, Firefox หรือ Safari (แนะนำ Chrome เวอร์ชันล่าสุด)

## 4. หน้ารายการรายงาน (Dashboard)

### 4.1 ภาพรวม

หน้า Dashboard เป็นหน้าแรกที่แสดงเมื่อเปิดแอปพลิเคชัน แสดงข้อมูลเชิงสรุปและรายการงานทั้งหมด

### 4.2 การ์ดสถิติ (Stat Cards)

ด้านบนของหน้าจะแสดงการ์ดสถิติ 4 ใบ:

| การ์ด | ความหมาย |
|---|---|
| 📁 **งานทั้งหมด** | จำนวนรายการงานทั้งหมดในระบบ |
| ⏳ **งานที่รอออกเลข** | งานที่ยังไม่มีเลขที่ประมาณการ |
| ✅ **งานที่ออกเลขแล้ว** | งานที่มีเลขที่ประมาณการแล้ว |
| 🖨️ **งานที่รอปริ้น** | งานที่ออกเลขแล้วแต่ยังไม่ได้สั่งพิมพ์รายงาน |

> [!NOTE]
> ตัวเลขสถิติจะมีแอนิเมชันนับขึ้นเมื่อโหลดหน้า เพื่อให้ผู้ใช้เห็นการเปลี่ยนแปลงชัดเจน

### 4.3 การค้นหา

- พิมพ์คำค้นหาในช่อง **"ค้นหาชื่อ, เลขประมาณการ, รหัสงาน..."**
- ระบบจะกรองรายการแบบ Real-time ขณะที่พิมพ์
- สามารถค้นหาด้วย: ชื่อลูกค้า, เลขประมาณการ, รหัสงาน (Job ID), หรือข้อมูลอื่นๆ

### 4.4 การจัดเรียง (Sort)

คลิกที่หัวคอลัมน์ของตารางเพื่อจัดเรียง:

| คอลัมน์ | การจัดเรียง |
|---|---|
| **รหัส / วันที่** | เรียงตาม Timestamp หรือ Job ID |
| **ลูกค้า / สถานที่** | เรียงตามชื่อลูกค้า (ก-ฮ) |
| **เลขประมาณการ** | เรียงตามเลขที่ประมาณการ |
| **ยอดรวม** | เรียงตามยอดเงินรวม (น้อย-มาก หรือ มาก-น้อย) |

> [!TIP]
> คลิกหัวคอลัมน์เดิมซ้ำอีกครั้งเพื่อสลับระหว่างเรียงจากน้อยไปมาก (↑) และมากไปน้อย (↓)

### 4.5 สถานะเลขประมาณการ (Badge)

แต่ละรายการงานจะแสดง Badge ตามสถานะ:

| Badge | สี | ความหมาย |
|---|---|---|
| 🕐 **รอออกเลข** | 🟡 เหลือง | ยังไม่มีเลขประมาณการ |
| ✅ **WN67/xxx** | 🟢 เขียว | มีเลขประมาณการแล้ว แต่ยังไม่ได้ปริ้น |
| 🖨️ **WN67/xxx** | 🔵 ฟ้า | มีเลขประมาณการและปริ้นแล้ว |

### 4.6 การจัดการรายการงาน

สำหรับแต่ละรายการงาน จะมีปุ่ม 2 ปุ่ม:

- 🖊️ **แก้ไข** — เปิดหน้า Form เพื่อแก้ไขข้อมูลงาน
- 🗑️ **ลบ** — ลบรายการงาน (จะมีกล่องยืนยันก่อนลบ)

> [!CAUTION]
> การลบรายการงานจะไม่สามารถกู้คืนได้ กรุณาตรวจสอบให้แน่ใจก่อนกดยืนยัน

### 4.7 การเพิ่มงานใหม่

คลิกปุ่ม **"+ เพิ่มงานใหม่"** ที่มุมขวาบนของหน้า Dashboard เพื่อไปยังหน้า Form สำหรับสร้างรายการงานใหม่

### 4.8 การรีเฟรชข้อมูล

คลิกปุ่ม 🔄 (หมุน) ที่แถบค้นหาเพื่อดึงข้อมูลล่าสุดจากฐานข้อมูล

> [!NOTE]
> ระบบมีกลไก **Cache** เพื่อโหลดข้อมูลเร็วขึ้น โดยจะดึงข้อมูลจาก Cache ก่อน แล้วอัปเดตจากฐานข้อมูลในพื้นหลัง หากต้องการข้อมูลล่าสุดทันที ให้กดรีเฟรช

## 5. หน้าบันทึกข้อมูลงานติดตั้ง (Form)

หน้า Form เป็นส่วนสำคัญที่สุดของระบบ ใช้สำหรับบันทึกข้อมูลงานติดตั้งประปาทั้งหมด แบ่งเป็น 5 ส่วน:

### 5.1 ส่วนที่ 1 — ข้อมูลเอกสาร

| ช่องข้อมูล | คำอธิบาย | ตัวอย่าง |
|---|---|---|
| **รหัสงาน (Job ID)** | รหัสเฉพาะของงาน สร้างอัตโนมัติ (ไม่สามารถแก้ไขได้) | `JOB-1740567890123` |
| **เลขที่คำขอ (Request No.)** | เลขที่คำขอรับบริการ | `RQ67/001` |
| **เลขที่ประมาณการ** | เลขที่ใบประมาณการ (กรอกเมื่อได้รับอนุมัติ) | `WN67/001` |
| **วันที่ขอรับบริการ** | วันที่ลูกค้ายื่นคำขอ (ค่าเริ่มต้นคือวันปัจจุบัน) | `2569-02-25` |

> [!IMPORTANT]
> **เลขที่ประมาณการ** เมื่อกรอกแล้ว ปุ่ม "ออกรายงาน" จะปรากฏขึ้นโดยอัตโนมัติ หากยังไม่ได้กรอก ปุ่มจะถูกซ่อนไว้

### 5.2 ส่วนที่ 2 — ข้อมูลลูกค้า

| ช่องข้อมูล | คำอธิบาย |
|---|---|
| **ชื่อ-สกุล ผู้ขอรับบริการ** | ชื่อเต็มของลูกค้า |
| **เลขที่ผู้ใช้น้ำ** | หมายเลขผู้ใช้น้ำประปา |
| **บ้านเลขที่** | บ้านเลขที่ของสถานที่ติดตั้ง |
| **หมู่** | หมู่บ้าน |
| **ตำบล** | ตำบล |
| **อำเภอ** | อำเภอ |
| **จังหวัด** | จังหวัด (ค่าเริ่มต้น: พระนครศรีอยุธยา) |

### 5.3 ส่วนที่ 3 — แผนที่และรูปแบบ

ส่วนนี้แบ่งเป็น 3 คอลัมน์:

#### 🗺️ พิกัดสังเขป (GPS)

ใช้สำหรับระบุพิกัดสถานที่ติดตั้งบนแผนที่:

- **วิธีที่ 1: คลิกปักหมุดบนแผนที่** — คลิกตำแหน่งที่ต้องการบนแผนที่โดยตรง ระบบจะบันทึกพิกัดโดยอัตโนมัติ
- **วิธีที่ 2: ใช้ GPS อุปกรณ์** — คลิกปุ่ม 🎯 (Crosshair) เพื่อดึงพิกัดจาก GPS ของอุปกรณ์
- **วิธีที่ 3: พิมพ์พิกัดด้วยตนเอง** — กรอกพิกัดในรูปแบบ `ละติจูด,ลองจิจูด` (เช่น `14.353200,100.569100`)

> [!TIP]
> หากใช้งานบนมือถือ การกดปุ่ม GPS จะช่วยให้ได้ตำแหน่งที่แม่นยำมากขึ้น ขอให้อนุญาตการเข้าถึงตำแหน่ง (Location Permission) ก่อนใช้งาน

#### 🖼️ ภาพจาก GIS

ใช้สำหรับอัปโหลดภาพจากระบบ GIS:

1. คลิกปุ่ม **"เลือกไฟล์"** หรือลากไฟล์มาวาง
2. เลือกไฟล์รูปภาพ (รองรับ JPG, PNG, WebP)
3. ระบบจะอัปโหลดรูปภาพไปยัง Supabase Storage โดยอัตโนมัติ
4. เมื่ออัปโหลดสำเร็จ จะแสดงภาพตัวอย่างในกรอบ

> [!WARNING]
> ขนาดไฟล์สูงสุด **5 MB** ต่อรูปภาพ หากเกินจะแจ้งเตือนและไม่สามารถอัปโหลดได้

#### 🔧 รูปแบบการติดตั้ง

เลือกรูปแบบมาตรฐานการติดตั้ง:

1. คลิกปุ่ม **"เลือกรูป"**
2. ระบบจะแสดงรายการรูปแบบ เช่น:
   - แบบมาตรฐาน 1 (ท่อลอย)
   - แบบมาตรฐาน 2 (ฝังดิน)
   - แบบมาตรฐาน 3 (ท่อลอย)  
   - แบบมาตรฐาน 4 (ท่อลอย)
   - ตัวอย่างหน้างาน
3. คลิกเลือกรูปแบบที่ต้องการ — ระบบจะแสดงตัวอย่างรูปภาพ

### 5.4 ส่วนที่ 4 — รายการอุปกรณ์ติดตั้ง

เป็นส่วนที่สำคัญที่สุดสำหรับการคำนวณค่าใช้จ่าย

#### การเลือกขนาดมิเตอร์

ก่อนเพิ่มรายการ ให้เลือกขนาดมิเตอร์:
- **1/2"** — ราคาเหมาจ่าย 3,600 บาท
- **3/4"** — ราคาเหมาจ่าย 4,700 บาท

> [!IMPORTANT]
> การเลือกขนาดมิเตอร์จะโหลดรายการอุปกรณ์เริ่มต้นโดยอัตโนมัติ (Default Items) สำหรับแต่ละขนาด

#### โครงสร้าง 8 ส่วนในรายการ

| ส่วนที่ | ชื่อ | คำอธิบาย |
|---|---|---|
| **1** | ส่วนติดตั้งมาตรวัดน้ำ | วัสดุและค่าแรงในการติดตั้งมาตรวัดน้ำ |
| **2** | ส่วนค่าติดตั้งเหมาจ่าย | ค่าใช้จ่ายแบบเหมาจ่ายสำหรับการติดตั้ง |
| **เหมาจ่าย** | (แถบรวม) | ยอดรวมเหมาจ่ายตามขนาดมิเตอร์ (3,600/4,700 บาท) |
| **3** | ส่วนวางท่อภายใน | วัสดุ ค่าแรง +เผื่อ 10% สำหรับท่อภายในบ้าน |
| **4** | ส่วนวางท่อภายนอก | วัสดุ ค่าแรง +เผื่อ 10% สำหรับท่อภายนอกบ้าน |
| **5** | ค่าดำเนินการ Factor F | คำนวณ = 1.3642 × (ส่วน 3 + ส่วน 4) |
| **6** | ค่าสำรวจ | คำนวณ = 2% × (ส่วน 3 + ส่วน 4) |
| **7** | ค่าธรรมเนียมประสานท่อ | ค่าธรรมเนียมเพิ่มเติม |
| **8** | เงินประกันมาตรวัดน้ำ | เงินประกันการใช้มาตรวัดน้ำ |

#### วิธีจัดการรายการ

- **เพิ่มรายการ:** ค้นหาชื่อวัสดุในช่องค้นหาของแต่ละส่วน แล้วคลิกเลือก
- **ปรับจำนวน:** กดปุ่ม **+** หรือ **−** เพื่อเพิ่มหรือลดจำนวน
- **ลบรายการ:** กดปุ่ม ❌ ที่ด้านขวาของรายการ
- **ดูยอดรวม:** ด้านล่างสุดจะแสดง **"รวมเป็นเงินทั้งสิ้น ... บาท"**

> [!NOTE]
> ส่วนที่ 3 และ 4 จะแสดงรายละเอียดการคำนวณเพิ่มเติม ได้แก่ ยอดวัสดุ, +เผื่อ 10%, และค่าแรง แยกออกจากกัน

### 5.5 ส่วนที่ 5 — การอนุมัติและบุคลากร

จัดการคณะผู้รับผิดชอบ 3 ตำแหน่ง:

| ตำแหน่ง | Badge สี | หน้าที่ |
|---|---|---|
| **ผู้สำรวจ** | 🔵 น้ำเงิน | ผู้ดำเนินการสำรวจพื้นที่ |
| **ผู้ตรวจสอบ** | 🟢 เขียว | ผู้ตรวจสอบความถูกต้องของข้อมูล |
| **ผู้อนุมัติ** | 🟡 เหลือง | ผู้มีอำนาจอนุมัติงาน |

#### วิธีเลือกบุคลากร

1. พิมพ์ชื่อในช่อง หรือเลือกจากรายชื่อที่ปรากฏ (Datalist)
2. เมื่อเลือกแล้ว ระบบจะ:
   - แสดง **ลายเซ็น** ของบุคคลนั้นโดยอัตโนมัติ (หากมี)
   - กรอก **ตำแหน่ง** ให้โดยอัตโนมัติ
3. สามารถพิมพ์ชื่อใหม่ได้หากบุคลากรไม่อยู่ในระบบ

### 5.6 การบันทึกข้อมูล

- **บนคอมพิวเตอร์:** คลิกปุ่ม **"💾 บันทึกข้อมูล"** ที่ด้านล่างขวา
- **บนมือถือ:** กดปุ่ม **"💾 บันทึก"** ที่แถบด้านล่างหน้าจอ (Fixed Bottom Bar)
- คลิกปุ่ม **"ยกเลิก"** หรือ **"← ย้อนกลับ"** เพื่อกลับหน้า Dashboard โดยไม่บันทึก

> [!CAUTION]
> กรุณาบันทึกข้อมูลก่อนออกจากหน้า Form มิฉะนั้นข้อมูลที่กรอกจะสูญหาย

### 5.7 โหมดสร้างใหม่ vs แก้ไข

| คุณสมบัติ | สร้างใหม่ (New) | แก้ไข (Edit) |
|---|---|---|
| **Job ID** | สร้างอัตโนมัติ (`JOB-xxxxx`) | แสดงจาก Job เดิม (ไม่แก้ไขได้) |
| **Mode Label** | "สร้างรายการใหม่ (JOB-xxxxx)" | "แก้ไขรายการ: JOB-xxxxx" |
| **ปุ่มออกรายงาน** | ซ่อน | แสดง (เมื่อมีเลขประมาณการ) |
| **สถานะการปริ้น** | ไม่แสดง | แสดงวันที่ปริ้นล่าสุด (ถ้ามี) |

## 6. หน้าคลังวัสดุ (Materials)

### 6.1 ภาพรวม

หน้าคลังวัสดุใช้สำหรับจัดการรายการวัสดุ/อุปกรณ์ทั้งหมดที่ใช้ในระบบ พร้อมราคาวัสดุและค่าแรง

### 6.2 การค้นหาและกรอง

- **ค้นหา:** พิมพ์ชื่อ, รหัส, หรือขนาดวัสดุ ในช่องค้นหา
- **กรองหมวดหมู่:** เลือกจาก Dropdown ได้แก่:

| หมวดหมู่ | คำอธิบาย |
|---|---|
| 1. ส่วนติดตั้งมาตรวัดน้ำ | วัสดุสำหรับติดตั้งมาตรวัดน้ำ |
| 2. ส่วนค่าติดตั้งเหมาจ่าย | รายการค่าเหมาจ่าย |
| 3. ส่วนวางท่อภายใน | ท่อและอุปกรณ์สำหรับภายในอาคาร |
| 4. ส่วนวางท่อภายนอก | ท่อและอุปกรณ์สำหรับภายนอกอาคาร |
| 7. ส่วนค่าธรรมเนียมประสานท่อ | ค่าธรรมเนียมต่างๆ |
| 8. เงินประกันมาตรวัดน้ำ | เงินประกัน |

### 6.3 การเพิ่มวัสดุใหม่

1. คลิกปุ่ม **"+ เพิ่มรายการ"**
2. กรอกข้อมูลใน Modal:

| ช่องข้อมูล | คำอธิบาย | ตัวอย่าง |
|---|---|---|
| **ชื่อรายการ** | ชื่อวัสดุ/อุปกรณ์ | ท่อ PVC ชั้น 8.5 |
| **รหัส (ID)** | รหัสวัสดุ (สร้างอัตโนมัติหากเว้นว่าง) | MAT-000001 |
| **ขนาด** | ขนาดของวัสดุ | 1/2" |
| **หน่วย** | หน่วยนับ (อัน, ชิ้น, ชุด, เมตร, ท่อน, เส้น, กล่อง, เครื่อง) | อัน |
| **หมวดหมู่** | เลือกหมวดหมู่จาก Dropdown | 3. ส่วนวางท่อภายใน |
| **ราคาวัสดุ** | ราคาวัสดุต่อหน่วย (บาท) | 45.00 |
| **ค่าแรง** | ค่าแรงต่อหน่วย (บาท) | 15.00 |

3. คลิก **"บันทึก"** เพื่อบันทึกรายการ

### 6.4 การแก้ไขวัสดุ

- คลิกปุ่ม 🖊️ **แก้ไข** ที่รายการที่ต้องการ
- แก้ไขข้อมูลใน Modal แล้วกด **"บันทึก"**
- รหัสวัสดุ (Material ID) ไม่สามารถแก้ไขได้ในโหมดแก้ไข

### 6.5 การลบวัสดุ

- คลิกปุ่ม 🗑️ **ลบ** ที่รายการที่ต้องการ
- ยืนยันการลบในกล่องข้อความ

### 6.6 การจัดเรียง

คลิกที่หัวคอลัมน์เพื่อจัดเรียง:
- **รหัส** — เรียงตามรหัสวัสดุ
- **ชื่อรายการ** — เรียงตามชื่อ (ก-ฮ)
- **หมวดหมู่** — เรียงตามหมวดหมู่
- **ราคาวัสดุ** — เรียงตามราคา
- **ค่าแรง** — เรียงตามค่าแรง

## 7. หน้าจัดการบุคลากร (Personnel)

### 7.1 ภาพรวม

จัดการข้อมูลบุคลากรที่มีส่วนเกี่ยวข้องกับงานประมาณการ ได้แก่ ผู้สำรวจ ผู้ตรวจสอบ และผู้อนุมัติ

### 7.2 การค้นหาและกรอง

- **ค้นหา:** พิมพ์ชื่อหรือตำแหน่งในช่องค้นหา
- **กรองบทบาท:** เลือกจาก Dropdown:
  - **ทุกบทบาท** — แสดงทั้งหมด
  - **ผู้สำรวจ (Surveyor)** — เฉพาะผู้สำรวจ
  - **ผู้ตรวจสอบ (Inspector)** — เฉพาะผู้ตรวจสอบ
  - **ผู้อนุมัติ (Approver)** — เฉพาะผู้อนุมัติ

### 7.3 การเพิ่มบุคลากรใหม่

1. คลิกปุ่ม **"👤+ เพิ่มบุคลากร"**
2. กรอกข้อมูลใน Modal:

| ช่องข้อมูล | คำอธิบาย | ตัวอย่าง |
|---|---|---|
| **ชื่อ-สกุล** | ชื่อเต็มของบุคลากร | นายสมชาย ใจดี |
| **ตำแหน่ง** | ตำแหน่งในองค์กร | ผู้จัดการ |

| **บทบาท (Role Type)** | เลือก 1 ใน 3 บทบาท | ผู้สำรวจ / ผู้ตรวจ / ผู้อนุมัติ |

3. เลือก **บทบาท** โดยคลิกที่การ์ด (มีไอคอนประกอบ)
4. คลิก **"บันทึก"**

> [!TIP]

- **ลบ:** คลิก 🗑️ ที่รายการ → ยืนยัน → ลบ

- **บทบาท** — เรียงตามบทบาท

## 8. ระบบรายงาน (Reports)

### 8.1 ภาพรวม

ระบบสามารถออกรายงาน 3 ฉบับพร้อมกัน:

| รายงาน | ชื่อ | คำอธิบาย |
|---|---|---|
| **R1** | ใบประมาณการ | แสดงรายละเอียดวัสดุ ราคา และยอดรวม |
| **R2** | ใบแจ้งค่าใช้จ่าย | แสดงค่าใช้จ่ายสรุปสำหรับลูกค้า |
| **R3** | แบบแปลน | แสดงแผนผังการติดตั้งพร้อมรูปภาพ |

### 8.2 วิธีออกรายงาน

1. เปิดหน้า Form ของงานที่ต้องการ (แก้ไขจาก Dashboard)
2. กรอก **เลขที่ประมาณการ** ในช่อง "เลขที่ประมาณการ"
3. ปุ่ม **"📄 ออกรายงาน"** จะปรากฏขึ้น
4. คลิกปุ่ม **"📄 ออกรายงาน"**
5. ระบบจะ:
   - เปิดหน้าต่างใหม่ 3 หน้าต่าง (R1, R2, R3) พร้อมกัน
   - บันทึกสถานะ "ปริ้นแล้ว" พร้อมวันที่ปริ้น
   - แสดงข้อความยืนยัน "📄 ออกรายงาน 3 ฉบับเรียบร้อย"

> [!WARNING]
> ตรวจสอบให้แน่ใจว่าเบราว์เซอร์ไม่ได้บล็อก Pop-up หากรายงานไม่เปิดขึ้นมา ให้อนุญาต Pop-up สำหรับเว็บไซต์นี้ในการตั้งค่าเบราว์เซอร์

### 8.3 สถานะการปริ้น

- หลังจากออกรายงาน จะแสดง **ไอคอน 🖨️** พร้อมวันที่ปริ้น ที่แถบด้านบนของ Form
- หากต้องการ **รีเซ็ตสถานะการปริ้น** ให้คลิกปุ่ม 🔄 ข้างวันที่ปริ้น (จะมีกล่องยืนยัน)

### 8.4 เคล็ดลับการพิมพ์

- เมื่อเปิดหน้ารายงานแล้ว ใช้ `Ctrl + P` (Windows) หรือ `Cmd + P` (Mac) เพื่อสั่งพิมพ์
- แนะนำเลือกกระดาษ **A4** และ **แนวตั้ง (Portrait)** สำหรับ R1 และ R2
- ปิด **Header/Footer** ของเบราว์เซอร์เพื่อให้รายงานสวยงาม

## 9. สูตรการคำนวณ

### สูตรคำนวณค่าใช้จ่ายงานติดตั้ง

```
ราคาเหมาจ่าย (Lump Sum)
├── มิเตอร์ 1/2" = 3,600 บาท
└── มิเตอร์ 3/4" = 4,700 บาท

ส่วนท่อภายใน/ภายนอก (ส่วน 3 และ 4)
├── ราคาต่อรายการ = ((ราคาวัสดุ × 1.10) + ค่าแรง) × จำนวน
├── เผื่อวัสดุ 10% = ราคาวัสดุรวม × 0.10
└── รวมส่วน = ราคาวัสดุรวม + เผื่อ 10% + ค่าแรงรวม

ค่าดำเนินการ Factor F (ส่วน 5)
└── = (ส่วน 3 + ส่วน 4) × 1.3642

ค่าสำรวจ (ส่วน 6)
└── = (ส่วน 3 + ส่วน 4) × 0.02

ยอดรวมค่าก่อสร้าง
└── = เหมาจ่าย + Factor F + ค่าสำรวจ + ค่าธรรมเนียม(ส่วน 7)

รวมเป็นเงินทั้งสิ้น (Grand Total)
└── = ยอดรวมค่าก่อสร้าง + เงินประกัน(ส่วน 8)
```

> [!NOTE]
> ทุกจำนวนเงินจะถูก **ปัดเศษเป็นจำนวนเต็ม** (Math.round) ก่อนรวมยอด

## 10. คำถามที่พบบ่อย (FAQ)

### ❓ ถาม: ทำไมข้อมูลไม่อัปเดตหลังจากแก้ไข?
**ตอบ:** ระบบใช้ Cache เพื่อโหลดเร็วขึ้น ให้กดปุ่ม 🔄 **รีเฟรช** ที่หน้า Dashboard เพื่อดึงข้อมูลล่าสุด

### ❓ ถาม: เลขที่ประมาณการต้องกรอกอย่างไร?
**ตอบ:** กรอกตามรูปแบบที่กำหนด เช่น `WN67/001` โดยระบบจะแสดงปุ่มออกรายงานเมื่อกรอกเลขนี้แล้ว

### ❓ ถาม: ทำไมปุ่ม "ออกรายงาน" ไม่แสดง?
**ตอบ:** ปุ่มนี้จะแสดงเฉพาะเมื่อ:
1. อยู่ในโหมดแก้ไข (ไม่ใช่สร้างงานใหม่)
2. ช่อง "เลขที่ประมาณการ" มีค่าที่ไม่ว่างเปล่า

### ❓ ถาม: รายงานไม่เปิดขึ้นมา (Pop-up Blocked)?
**ตอบ:** ให้อนุญาต Pop-up ในเบราว์เซอร์:
- **Chrome:** คลิกไอคอน Pop-up ที่แถบ URL → อนุญาตเสมอ
- **Edge:** ตั้งค่า → ความเป็นส่วนตัว → อนุญาต Pop-up

### ❓ ถาม: อัปโหลดรูป GIS ไม่ได้?
**ตอบ:** ตรวจสอบ:
1. ขนาดไฟล์ไม่เกิน 5 MB
2. ไฟล์เป็นรูปภาพ (JPG, PNG, WebP)
3. มีการเชื่อมต่ออินเทอร์เน็ต

### ❓ ถาม: แผนที่ไม่โหลด?
**ตอบ:** แผนที่ต้องอาศัยอินเทอร์เน็ต (OpenStreetMap) ตรวจสอบการเชื่อมต่อ และลอง Refresh หน้าเว็บอีกครั้ง

### ❓ ถาม: ลายเซ็นบุคลากรไม่แสดง?
**ตอบ:** ตรวจสอบว่าได้กรอก URL ลายเซ็นไว้ในหน้าจัดการบุคลากร และ URL นั้นสามารถเข้าถึงได้ (ไม่ถูกบล็อกโดย CORS)

### ❓ ถาม: ข้อมูลหายหลังจากปิดเบราว์เซอร์?
**ตอบ:** ข้อมูลทั้งหมดถูกบันทึกไว้ในฐานข้อมูล Supabase บนคลาวด์ เพียงแค่เปิดเว็บแอปอีกครั้งก็สามารถเข้าถึงข้อมูลได้ตามปกติ ข้อมูลจะไม่สูญหายตราบใดที่กดปุ่ม "บันทึก" แล้ว
